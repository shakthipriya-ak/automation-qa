# Day 3 - Strings and Pattern Printing

## Topics Covered

* String Indexing
* Negative Indexing
* String Slicing
* String Methods

  * upper()
  * lower()
  * title()
  * replace()
  * split()
  * strip()
  * find()
* Nested Loops
* Pattern Printing

## Patterns Practiced

* Square Pattern
* Left Triangle
* Reverse Triangle
* Number Triangle
* Increasing Number Pattern
* Half Diamond
* Number Half Diamond
* Reverse Number Triangle

## Key Learnings

* Strings are immutable.
* Slicing follows the format string[start:end:step].
* Outer loop controls the number of rows.
* Inner loop controls what gets printed in each row.
* Complex patterns can be broken into smaller simple patterns.
* Pattern solving becomes easier by identifying:

  * Number of rows
  * Number of spaces
  * Number of symbols to print

## Important Concept

Before writing any pattern, ask:

1. How many rows are there?
2. How many items should be printed in this row?
3. What should be printed?
4. Does the count increase or decrease?
