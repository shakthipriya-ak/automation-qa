## Key Learnings

- Functions help divide a program into reusable blocks.
- `return` sends values back to the caller.
- `global` allows modification of global variables inside functions.
- Default arguments provide fallback values.
- Keyword arguments improve readability.
- `*args` accepts multiple positional arguments.
- `**kwargs` accepts multiple keyword arguments.
- Lambda functions are useful for simple one-line functions.
- Modules help organize and reuse code.
- Python provides many built-in modules for common tasks.
- A class is a blueprint, and an object is an instance created from that blueprint.
